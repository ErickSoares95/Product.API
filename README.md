# WEB API - ASP .NET Core 5.0

Objetivo desse projeto é se familiarizar mais com a tecnologia  ASP.NET Core 5.0. Foi utilizada a abordagem  code-first para gerenciar a base de dados MySql.

### Recursos utilizados

- ASP .NET Core 5.0
- EF Core 5.0.1
- MySql 8.0​

### Primeiros passos 
Ao clonar o repositório deve-se carregar o arquivo `Product.sln` na sua IDE. Uma vez que os pacotes NuGet da solução forem carregados, basta modificar a ConnectionStrings no arquivo `appsettings.json` para executar o projeto.
